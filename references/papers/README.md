# PapiMiner Paper Shelf

This folder is only a bibliography for papers used during PapiMiner research.
The PDFs themselves are ignored by git; link to original sources instead of
redistributing downloaded copies.

这个目录只放 PapiMiner 研究时用到的论文索引。PDF 本体被 git 忽略；公开仓库里建议链接原始来源，不重新分发下载副本。

## Topics

- Pearl protocol and PlainProof background
- PoUW and matrix-multiplication security assumptions
- GPU Tensor Core implementation ideas
- Public criticism and measurement of current Pearl deployments

## Release Rule

Keep this folder lightweight. Do not include local notes, copied PDFs, account
data, benchmark logs, wallet addresses, worker names, or machine-specific GPU
details.
